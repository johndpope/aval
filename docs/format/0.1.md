# Compiled format 0.1

An `.rma` asset has a fixed header, canonical front-index manifest and sample
table, aligned payload layout, independently decodable AVC units, and strict PNG
statics. All integer arithmetic, UTF-8, canonical JSON, offsets, overlaps,
geometry, AVC, PNG, and digest fields are validated before media use.

The graph supports authored states, body units, transitions, ports, portals,
finish/cut/locked routes, exact inverse pairs, bindings, readiness, and fallback
policy. Half-open frame ranges and rational frame rate define behavior; there is
no runtime seek instruction or embedded script.

Package `1.0.0` continues to read wire version `0.1`. Package semver and wire
version are independent compatibility spaces.
