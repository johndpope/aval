# Compiler project 0.2

Project `0.2` is strict JSON describing canvas, frame rate, bounded sources,
author-defined states, units, ports, edges, bindings, readiness, fallback,
renditions, alpha policy, and output limits. Unknown fields and ambiguous
timelines are rejected.

The compiler resolves source files relative to the project, never from asset
URLs. Reproducible builds require the complete recorded Node, compiler,
FFmpeg/FFprobe executable, configuration, encoder, and calibration fingerprint.
Another supported tool build may be semantically conformant without producing
the same AVC bytes.
