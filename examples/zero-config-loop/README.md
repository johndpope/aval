# Zero-configuration loop illustration

This source example is illustrative: the `orbit` codec bundle and `orbit.png`
are placeholders and are not included here. Supply those files (or change the
paths), copy the exact MIME types and per-file integrity values from the
bundle's `build.json`, then run `npm install` and `npm run build`.

For an immediately runnable generated asset and browser workflow, use
`avl init` and the checked starter instead.
