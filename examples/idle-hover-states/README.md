# Idle/hover states illustration

This source example is illustrative: the `favorite` codec bundle is a
placeholder and is not included here. Supply a bundle with the authored
`idle` and `selected` states (or change the code), copy the exact MIME types
and per-file integrity values from `favorite/build.json`, then run
`npm install` and `npm run build`.

For an immediately runnable generated idle/engaged asset and browser workflow,
use `avl init` and the checked starter instead.
