# Network and integrity illustration

This source example is intentionally illustrative. The `.example.test` origin,
asset, fallback image, and integrity token are placeholders; it must not be
described as a live endpoint. Replace them with one immutable hosted asset and
its real SHA-256 token before running `npm install` and `npm run build`.

For an immediately runnable local workflow, use `rma init` and `npm run dev` in
the generated starter.
